\# South Korea Top 50 Playlist Analysis



\## Project Overview



This project analyzes comeback momentum, chart re-entry behavior, and fandom intensity within the South Korea Top 50 Playlist dataset.



The objective is to identify songs that repeatedly return to the charts, measure comeback strength, and evaluate fandom-driven engagement patterns.



\## Features



\* Re-entry Frequency Analysis

\* Popularity Distribution

\* Album vs Single Analysis

\* Explicit vs Non-Explicit Analysis

\* Artist Leaderboard

\* Interactive Dashboard



\## Technologies Used



\* Python

\* Pandas

\* Streamlit

\* Plotly



\## Dataset



South Korea Top 50 Playlist Dataset containing:



\* Date

\* Position

\* Song

\* Artist

\* Popularity

\* Album Type

\* Explicit Content

\* Duration
